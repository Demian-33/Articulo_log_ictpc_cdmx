# Articulo log-normal sesgado ictpc cdmx
Este repositorio contiene:

- Código fuente de R, Stan y conjunto de datos para reproducir el análisis estadístico. Para instalar Stan, siga las instrucciones en *https://mc-stan.org/cmdstanr/*. El conjunto de datos se dispone en formato `.csv` y también como objeto de R (`.RData`) para facilidad en el análisis.


- Código LaTeX para generar un `.pdf` con el documento.

Para generar un documento MS Word (`.docx`), puede emplearse `pandoc`.
